# Дизайн-система — Деревня Знаний

Цель: тёплый, дружелюбный, не «детсадовский». Подходит детям 6–11 и не выглядит дешёво в глазах родителя.

## Палитра

| Токен | HEX | Использование |
|---|---|---|
| `--background` | `#FFF9F0` | Фон страниц |
| `--foreground` | `#1F2937` | Основной текст |
| `--primary` | `#4DA8DA` | Главные действия |
| `--primary-foreground` | `#FFFFFF` | Текст на primary |
| `--accent` | `#FFB347` | Награды, монетки |
| `--success` | `#6BCB77` | Правильный ответ, прогресс |
| `--danger` | `#FF6B6B` | Ошибка |
| `--muted` | `#F1ECE2` | Карточки |
| `--border` | `#E5DFD2` | Бордеры |

В Tailwind v4 — через `@theme` в `globals.css`. shadcn-токены замапить на эти.

## Типографика

- **Шрифт UI**: Nunito (Google Fonts), веса 400/600/800.
- **Шрифт игровой**: тот же Nunito, но в R3F через drei `<Text font={...}>` подгружаем wof2.
- Шкала: 12 / 14 / 16 (base) / 18 / 22 / 28 / 36.
- Заголовки — bold (800), обычный текст 400, кнопки 600.

## Радиусы и тени

- Радиус карточек: `1rem` (16px).
- Радиус кнопок: `0.75rem`.
- Тень для карточек: `0 6px 20px rgba(31,41,55,0.08)`.
- В детской игре — чуть мягче, в дашборде — чуть строже.

## Размеры тач-целей

- Минимум **56×56 px** в детских экранах.
- В дашборде родителя — стандартные 40×40 шадсиновые.

## Иконография

- `lucide-react`. Размер 20–24 для дашборда, 28–32 для детских HUD.
- Для монеток — кастомная SVG-иконка (`/public/icons/coin.svg`), для энергии — `/public/icons/energy.svg`.

## Тон

- **Ребёнку**: короткие фразы, «Молодец!», «Попробуй ещё!», без насмешек. Никаких «неверно» — лучше «не угадал, попробуй ещё».
- **Родителю**: спокойный, информативный, без излишней теплоты.

## Анимация

- Использовать `framer-motion` точечно — для появления карточек, переходов модалок. В R3F — анимация через `useFrame` + lerp.
- В детских экранах после правильного ответа — короткая «искра» (CSS keyframes на pseudo-element).

## Компоненты shadcn для установки

`npx shadcn@latest add button card dialog tabs input label form toast avatar badge progress separator scroll-area dropdown-menu sheet`

## Скриншоты-референсы

- Дашборд: вдохновение — Linear, Notion-простые карточки.
- Детский UI: вдохновение — Duolingo (большие кнопки, пастель), Khan Kids (мягкие тени).
- 3D-мир: вдохновение — Animal Crossing Mobile, Hay Day (изометрия, мягкие цвета).

Никаких лицензированных персонажей — все спрайты и текстуры рисуются placeholder-ами и могут быть позже заменены оригинальными.
