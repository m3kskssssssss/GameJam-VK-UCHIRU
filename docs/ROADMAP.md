**DoD**: можно зарегистрировать родителя, добавить ребёнка, выйти, войти как ребёнок и оказаться на `/play`. И наоборот для родителя — `/parent`.

## Фаза 5 — Домики заданий (мини-игры в стиле Duolingo)

**Ведущий**: `architect` → **Исполнители**: `mini-game-developer`, `ui-engineer`, `backend-engineer`

Общий контракт:
- Вход в домик → экран «Уровень N — Начать». После завершения уровня → «Перейти к следующему уровню» / «Выйти из домика».
- Все ответы валидируются мгновенно по эталону (никаких AI). Контент генерируется детерминированно из сидов.
- За пройденный уровень: +N монет, +N энергии (см. `docs/GAME_DESIGN.md`).

### 5.1 Математика
- [ ] Уровни 1–10: сложение → вычитание → умножение → дроби. Уровень = 10 заданий.
- [ ] Типы: выбор из 4 вариантов, ввод числа.

### 5.2 Чтение (русский)
- [ ] Уровни 1–10: короткие тексты (50–200 слов) + 3 вопроса с выбором ответа.

### 5.3 Английский
- [ ] Уровни 1–10: словарь и простые фразы. Типы: «выбери перевод», «собери из слов», «слушай и выбери» (используем Web Speech API для TTS, бесплатно в браузере).

### 5.4 Физкультура (особый поток)
- [ ] Список упражнений (приседания, отжимания, прыжки и т.п.). Ребёнок выбирает упражнение → инструкция с GIF/иллюстрацией → «Я готов».
- [ ] **После «Я готов»** запрашивается доступ к камере (`getUserMedia`).
- [ ] **Через 10 секунд** — авто-снимок (без таймера на экране, чтобы не отвлекать). **Через 60 секунд** — второй авто-снимок. Затем «Завершить».
- [ ] Оба фото загружаются в Vercel Blob, создаётся `PESession` со ссылками. Ребёнку показывается «Молодец!» + награда.
- [ ] Никакой проверки качества — просто два снимка для родителя.

**DoD**: каждый из 4 домиков отдаёт результат в БД, копит прогресс и награды. У родителя в дашборде видны новые попытки и фото.

## Фаза 6 — Главный домик (Sims-like)

**Ведущий**: `architect` → **Исполнители**: `game-world-developer`, `ui-engineer`, `backend-engineer`

- [ ] Вход в главный домик → отдельная сцена: вид комнаты сверху (R3F или 2D Canvas — на усмотрение, но с тем же визуальным языком).
- [ ] **Гардероб**: смена внешности персонажа (волосы, верх, низ). Часть бесплатно, часть за монетки.
- [ ] **Магазин мебели**: каталог предметов (стол, стул, ковёр, лампа и т.п.). Покупка → предмет в инвентаре.
- [ ] **Расстановка**: перетаскивание купленной мебели в комнате (сетка). Сохранение позиций в БД.
- [ ] **Питомец**: купить одного из 3 питомцев, кормить за энергию (лёгкая механика).
- [ ] **Покупка следующей комнаты**: открывает `Room` с index=1, можно переключаться между комнатами.
- [ ] **Прокачка**: уровень дома повышается, когда сумма уровней по 4 предметам растёт. Визуально — больше декораций / новый фон.
- [ ] HUD-кнопка «Выйти к домикам заданий» возвращает в Фазу 4.
- [ ] HUD-кнопка «Сервер игроков» — переход в Фазу 7.

**DoD**: ребёнок может купить мебель и одежду за монетки, расставить, кастомизировать персонажа, разблокировать вторую комнату.
